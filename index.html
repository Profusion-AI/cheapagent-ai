        How it works
        
Profile first. Encode second. Measure always.

        
doc2toon does not claim savings before it proves them. CheapAgent exposes the measured result in the browser.

      


      

        

          
1
Read.md, .txt, paste
Markdown, plain text, or a pasted agent file. Bodies stay in the browser by default.

          
2
Profile the documentdetect structure
The profiler detects definitions, requirements, rules, and tables.

          
3
Encode to TOONbrowser-safe package
Repeated-record shapes get denser while the receipt stays reviewable.

          
4
Surface warningsreview before trust
Duplicate, vague, long, and split-candidate warnings are shown without rewriting silently.

          
5
Report measured metricsno guessing
Source chars, TOON chars, token estimates, and the real delta for this input.

        The claims boundary
        
Measure before you claim savings.

        
CheapAgent is an early alpha for browser-side agent-context measurement. Savings are measured per input and are not guaranteed.

      

      

        
ok
What it does
Measures structure, warnings, source size, output size, and a token estimate locally in the browser.

        
ok
Privacy default
The page does not call a hosted LLM API or store document bodies server-side.

        
no
What it will not claim
No universal percentage savings, no compliance assurance, no production-scale reliability claim, and no silent lossy rewrite.
